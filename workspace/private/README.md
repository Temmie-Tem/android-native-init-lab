# Private Workspace

This tree is the default local working area. Its payload is ignored by Git.
Only README files and optional `.gitkeep` placeholders are tracked.

Use this area for raw, private, large, or redistributability-unclear files.
Promote only redacted metadata or summaries to `../public/` or `../../docs/artifacts/`.

Generated boot-test outputs belong under `builds/`. Restored seed/current inputs
belong under `inputs/`.
